# Colors

## Semantic Roles (Recommended)

> Prefer these roles in components. Each role maps to a base palette token.
> Use the palette (`theme.hugoUIColors`) only as an escape hatch when a semantic role doesn't fit.
> Roles should not expose palette aliases (like `neutral.grey800`).

## Palette (Base Tokens)

### Primary Colors

> The primary color and its variants are used most frequently for important actions such as highlighting
> information, interacting with UI elements and adding personality to the brand.
